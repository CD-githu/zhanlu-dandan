# 快速启动指南

## 环境要求

### 必需软件
- **Node.js** 18.0+ (推荐 20.x)
- **npm** 9.0+ 或 **pnpm** 8.0+
- **Rust** 1.70+ (Tauri 需要)
- **Git**

### Windows 特别要求
- Visual Studio C++ Build Tools
- WebView2 (通常 Windows 11 自带)

### macOS 特别要求
- Xcode Command Line Tools

### Linux 特别要求
- libwebkit2gtk-4.0-dev
- build-essential
- curl
- wget
- file
- libssl-dev
- libgtk-3-dev
- libayatana-appindicator3-dev
- librsvg2-dev

---

## 环境安装

### 1. 安装 Node.js

```bash
# 下载：https://nodejs.org/
# 或使用 nvm (推荐)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20
nvm use 20
```

### 2. 安装 Rust

```bash
# Windows: 下载 rustup-init.exe
# https://www.rust-lang.org/tools/install

# macOS/Linux
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```

### 3. Windows 安装 C++ Build Tools

下载并安装：https://visualstudio.microsoft.com/visual-cpp-build-tools/

选择 "Desktop development with C++" 工作负载。

### 4. Linux 安装依赖

```bash
sudo apt update
sudo apt install libwebkit2gtk-4.0-dev \
  build-essential \
  curl \
  wget \
  file \
  libssl-dev \
  libgtk-3-dev \
  libayatana-appindicator3-dev \
  librsvg2-dev
```

## 数据库初始化

```bash
# 初始化 Prisma
npx prisma init --datasource-provider sqlite

# 配置 schema.prisma
cat > prisma/schema.prisma << 'EOF'
datasource db {
  provider = "sqlite"
  url      = "file:./workboard.db"
}

generator client {
  provider = "prisma-client-js"
}

model Task {
  id          String   @id @default(unique())
  title       String
  description String?
  type        String   // daily, temp, long
  priority    String   // high, medium, low
  status      String   @default("todo")
  tags        String   @default("")
  assignee    String?
  dueDate     DateTime?
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
  completedAt DateTime?
}

model Schedule {
  id          String   @id @default(unique())
  title       String
  description String?
  startTime   DateTime
  endTime     DateTime
  allDay      Boolean  @default(false)
  reminderTime Int?
  recurring   String?
  taskId      String?
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
EOF

# 运行迁移
npx prisma migrate dev --name init

# 生成 Prisma Client
npx prisma generate
```

## 常见问题

### 环境变量配置

创建 `.env` 文件：

```env
VITE_API_BASE_URL=http://localhost:3000
VITE_AI_API_URL=http://192.168.1.100:8000/v1
```

### 代理配置

如果需要代理 API，在 `vite.config.ts` 中添加：

```typescript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:3000',
      changeOrigin: true
    }
  }
}
```

### 端口冲突

如果端口 5173 被占用，修改 `vite.config.ts`：

```typescript
server: {
  port: 5174
}
```
