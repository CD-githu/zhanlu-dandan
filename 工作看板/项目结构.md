## 核心文件说明

### 前端核心文件

| 文件 | 说明 |
|------|------|
| `src/main.ts` | 应用入口，初始化Vue、路由、Store |
| `src/App.vue` | 根组件，包含整体布局 |
| `vite.config.ts` | 构建配置，别名、代理、插件 |
| `tsconfig.json` | TypeScript编译配置 |

### 后端核心文件

| 文件 | 说明 |
|------|------|
| `src-tauri/src/main.rs` | Rust应用入口 |
| `src-tauri/src/commands.rs` | Tauri命令定义，前后端通信接口 |
| `src-tauri/src/db.rs` | 数据库操作封装 |
| `src-tauri/Cargo.toml` | Rust依赖配置 |
| `src-tauri/tauri.conf.json` | Tauri应用配置（窗口、权限等）|

### 数据库文件

| 文件 | 说明 |
|------|------|
| `prisma/schema.prisma` | 数据模型定义 |
| `prisma/migrations/` | 迁移历史 |
| `workboard.db` | SQLite数据库文件（运行时生成）|

## 文件创建顺序建议

1. **基础配置文件**
   - package.json
   - vite.config.ts
   - tsconfig.json

2. **类型定义**
   - src/types/index.ts
   - src/types/task.ts
   - src/types/schedule.ts

3. **状态管理**
   - src/store/index.ts
   - src/store/task.ts
