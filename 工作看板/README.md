# 工作看板桌面应用

> 一个集成任务管理、日程安排、工作流程、计划总结和协作交接的桌面工作看板应用

---

## 📋 项目概述

工作看板是一个专为个人和小团队设计的桌面应用，集成了日常工作的所有核心功能：

- ✅ **任务管理** - 日常任务、临时任务、长期任务一体化管理
- 📅 **日程安排** - 日历视图、时间块管理、智能提醒
- 🔄 **工作流程** - 流程可视化、节点配置、标准化操作
- 📊 **计划总结** - 周月年度计划、AI辅助生成、数据分析
- 👥 **协作交接** - AB角权限管理、工作交接、操作日志

---

## 🎯 核心特性

### 1. 多维度任务管理
- 任务分类：日常 / 临时 / 长期
- 优先级设置：高 / 中 / 低
- 状态流转：待办 → 进行中 → 暂停 → 已完成
- 标签系统、子任务、附件支持

### 2. 灵活的日程安排
- 月/周/日多视图切换
- 时间块管理、冲突检测
- 桌面通知提醒
- 定期重复任务支持

### 3. 可视化工作流程
- 流程模板创建
- 节点可视化编辑
- 关键节点标注
- 流程统计分析

### 4. 智能计划总结
- AI自动生成工作总结（支持内网大模型）
- 数据可视化图表
- 完成度追踪
- 导出多格式文档

### 5. 协作与交接
- AB角权限管理
- 工作交接记录
- 操作日志审计
- 数据备份恢复

## 🛠️ 技术栈

### 前端
- **Vue 3** - 渐进式框架
- **TypeScript** - 类型安全
- **Vite** - 高性能构建
- **Element Plus** - UI组件库
- **Pinia** - 状态管理
- **Vue Router** - 路由管理

### 后端
- **Tauri 2.x** - 桌面应用框架
- **Rust** - 高性能后端
- **SQLite** - 本地数据库
- **Prisma ORM** - 数据库操作

### 工具库
- **dayjs** - 日期处理
- **nanoid** - ID生成
- **hotkeys-js** - 快捷键

## 🚀 快速开始

### 前置要求
- Node.js 18.0+
- Rust 1.70+
- (Windows) Visual Studio C++ Build Tools
- (macOS) Xcode Command Line Tools

### 安装运行

```bash
# 1. 安装依赖
npm install

# 2. 初始化数据库
npx prisma generate

# 3. 启动开发环境
npm run tauri:dev
```

### 构建发布

```bash
# 构建生产版本
npm run tauri:build

# 输出位置：src-tauri/target/release/bundle/
```

## 🔧 配置说明

### AI大模型配置

在设置中配置内网AI大模型：

```json
{
  "apiUrl": "http://192.168.1.100:8000/v1",
  "apiToken": "your_token_here",
  "model": "default_model"
}
```

### 数据备份配置

```json
{
  "autoBackup": true,
  "backupPath": "D:/backups/workboard",
  "interval": 24
}
```
